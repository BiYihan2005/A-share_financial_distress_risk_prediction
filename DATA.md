# Data Format

This repository does not include the original private/course dataset. The pipeline can be tested with synthetic data generated by `scripts/make_synthetic_data.py`.

## Expected Input

The full pipeline expects one CSV file:

```text
data/raw/financial_data.csv
```

The CSV should contain firm identifiers, categorical variables, target variables, and numeric financial indicators.

## Identifier and Categorical Columns

| Column | Meaning | Used as Model Feature |
|---|---|---|
| `股票代码` | Stock code | No |
| `股票简称` | Stock short name | No |
| `行业名称1` | Industry name | Yes, one-hot encoded |
| `报表类型编码` | Report type code | Yes, one-hot encoded |

## Target Variables

| Column | Meaning | Direction |
|---|---|---|
| `Oscore` | Main financial distress risk score | Higher means riskier |
| `Zscore` | Financial safety score | Higher usually means safer |
| `Zrisk` | Constructed as `-Zscore` | Higher means riskier |

The pipeline automatically creates `Zrisk` when `Zscore` is available.

## Main Financial Indicators

Typical financial indicators include:

| Column | Financial Meaning |
|---|---|
| `托宾Q值A` | Market valuation and growth expectation |
| `EVA率口径一` | Economic value creation after capital cost |
| `销售EVA率口径一` | Economic value added per unit of sales |
| `现金流到期债务保障倍数` | Operating cash flow coverage of maturing debt |
| `净资产收益率增长率A` | Change in return on equity |
| `营业利润增长率A` | Operating profit growth |
| `现金股利保障倍数` | Cash-flow coverage of dividend payments |
| `盈利波动性` | Earnings volatility and business uncertainty |
| `应付账款周转率A` | Supplier-credit and operating-liability management |
| `净利润现金净含量` | Cash-flow support behind accounting profit |
| `营运指数` | Quality of converting operating income into cash flow |
| `市盈率PE1` | Valuation and earnings expectation |

## Why the Real Dataset Is Not Included

Financial statement and course datasets may have redistribution restrictions. For this reason, `data/raw/`, `outputs/`, and `models/` are ignored by Git.

The synthetic dataset is only for verifying that the code runs end-to-end. It should not be interpreted as real company data.
