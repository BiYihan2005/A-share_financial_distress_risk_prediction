# 数据字段说明

公开仓库不包含原始真实数据。若要使用自己的数据运行完整流程，请准备一个 CSV 文件，并尽量保持以下字段名称一致。

## 1. 标识列和分类列

| 字段名 | 含义 | 是否进入模型 |
|---|---|---|
| 股票代码 | 公司股票代码 | 不作为数值特征，用于样本识别和分组统计 |
| 股票简称 | 公司简称 | 不作为数值特征，用于展示和追踪样本 |
| 行业名称1 | 公司所属行业 | 通过 One-Hot 编码进入模型 |
| 报表类型编码 | 财务报表类型 | 通过 One-Hot 编码进入模型 |

## 2. 被解释变量

| 字段名 | 含义 | 解释方向 |
|---|---|---|
| Oscore | 财务困境风险综合得分 | 通常越高表示风险越高 |
| Zscore | 财务安全性得分 | 通常越高表示越安全 |
| Zrisk | 由代码自动构造，`Zrisk=-Zscore` | 越高表示风险越高 |

## 3. 主要财务指标

| 字段名 | 财务含义 |
|---|---|
| 托宾Q值A | 市场估值与成长预期 |
| EVA率口径一 | 扣除资本成本后的价值创造能力 |
| 销售EVA率口径一 | 单位销售收入创造经济增加值的能力 |
| 现金流到期债务保障倍数 | 经营现金流对到期债务的覆盖能力 |
| 净资产收益率增长率A | 股东权益回报能力的变化 |
| 营业利润增长率A | 主营经营盈利能力变化 |
| 现金股利保障倍数 | 现金流对股利支付的保障能力 |
| 盈利波动性 | 盈利稳定性和经营不确定性 |
| 应付账款周转率A | 供应商信用和经营性负债管理 |
| 净利润现金净含量 | 会计利润的现金流支撑程度 |
| 营运指数 | 经营收益转化为现金流的质量 |
| 市盈率PE1 | 市场估值和盈利预期 |

## 4. 示例数据

`data/example/synthetic_financial_data.csv` 是合成示例数据，只用于测试代码能否运行。它不代表任何真实公司。
